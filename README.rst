
Python implementations of Sentinel 2 Toolbox L2 products & spectral indices

Installation
-------------

The sentipy package is built & published (with enormous gratitude) using `Poetry <https://python-poetry.org/>`_

.. code-block:: bash

    $ pip install sentipy
